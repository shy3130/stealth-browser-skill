All notable changes to this project will be documented in this file.

### Fixed
- Fixed Chrome launcher async/await syntax error
- Improved port availability checking

### Changed
- Updated import statements for better compatibility
- Enhanced Chrome detection logic

### Added
- Initial release
- Real Chrome-based anti-detection browser automation

- AI-friendly JavaScript API
- Chrome extension for enhanced anti-detection
